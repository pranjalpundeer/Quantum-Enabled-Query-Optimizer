
# ⚛️ Quantum Enabled Query Optimizer

A futuristic SQL query analyzer that bridges classical query optimization and quantum computing readiness.  
Built using **Python**, **Streamlit**, and a touch of **quantum-inspired logic**.

---

## 🚀 Features

- **Query Classification** → Detects whether a query is *Simple* or *Complex*  
- **Complexity Scoring** → Calculates a numerical score based on query features  
- **Feature Extraction** → Identifies joins, subqueries, aggregations, group by, having, and analytical functions  
- **Quantum Prep Stage** → Converts features into a *normalized quantum state vector*  
- **Qubit Estimation** → Predicts the approximate number of qubits required  
- **Modern UI** → A clean, dark-themed dashboard built with Streamlit  

---

## 🧠 Architecture Overview

1. **Phase 2A – Query Complexity Visualization**  
   Analyzes and classifies SQL queries with a modern Streamlit interface.  

2. **Phase 2B – Quantum Prep**  
   Converts extracted features into a quantum-ready normalized vector  
   and estimates resource requirements for quantum computation.  

---

## 🧩 Tech Stack

- **Python 3.10+**  
- **Streamlit** (UI)  
- **sqlparse** (SQL parsing)  
- **NumPy** (vector normalization)
